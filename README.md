# Financial-Research-and-Analysis
Financial Analysis: Finding Relationships between Financial and Non-Financial Aspect of Comapny

This project investigates whether AI-related investor attention drives speculative trading behaviour across equity markets, drawing parallels with the dot-com bubble of the late 1990s. Using daily trading volume data for 75 firms each from the Nasdaq-100 and FTSE-100 (sourced via the LSEG Eikon API) over the period 2019–2024, I engineered a monthly volume index through Z-score normalisation and equal-weighted aggregation to isolate abnormal trading activity. To capture investor attention, I constructed a composite AI Search Index from region-specific Google Trends data (keywords: Artificial Intelligence, Generative AI, AI Bubble, AI Trade, Tech Bubble) using Principal Component Analysis to address multicollinearity across variables. 
OLS time-series regression models, tested for autocorrelation via the Durbin-Watson statistic and adjusted using HAC standard errors — were fitted for both indices against their respective AI indices. 
Results show that AI-related public attention has a statistically significant negative relationship with Nasdaq-100 trading volume, while showing no significant effect on the FTSE-100, supporting the hypothesis that current AI speculation is firm-specific (concentrated in companies like Nvidia) rather than a broad market phenomenon. The project demonstrates applied skills in financial econometrics, time-series analysis, dimensionality reduction, and quantitative data sourcing.
